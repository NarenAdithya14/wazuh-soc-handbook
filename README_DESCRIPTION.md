This repository contains a collection of lab work and proof-of-concept exercises focused on the Wazuh Security Information and Event Management (SIEM) platform. It serves as a practical record of implementing and exploring various Wazuh capabilities, including log analysis, file integrity monitoring (FIM), threat intelligence integration, and attack detection. The labs cover fundamental setup, agent configuration, SOC dashboard creation, and simulated attack scenarios to test detection and response. It is intended as a reference for understanding Wazuh's core features in a hands-on context.
